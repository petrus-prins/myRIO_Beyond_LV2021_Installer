Getting Started with myRIO Programming

Let's learn from the following tutorials to get started with myRIO programming:

  • Creating Your First myRIO Application—Teaches you how to create a myRIO application to control the myRIO onboard devices.
  • Configuring a Wireless Network on the myRIO—Teaches you how to connect to and create a wireless network on the myRIO.

If you connect your myRIO to a host computer for the first time, run the Getting Started with myRIO wizard before learning these tutorials. When you run the Getting Started with myRIO wizard, LabVIEW installs the recommended software set on a new myRIO and tests the onboard devices. You can always launch the Getting Started with myRIO wizard by clicking the Set Up and Explore link in LabVIEW and selecting Launch the Getting Started Wizard.

You can always revisit these tutorials by clicking the Set Up and Explore link in LabVIEW and selecting Access Getting Started Tutorials.
